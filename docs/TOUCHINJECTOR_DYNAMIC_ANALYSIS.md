## 🔍 TouchInjector 사용 컨텍스트

### 1. TouchInjector 클래스

```java
// TouchInjector.java
package com.loveplusplus.update;

public class TouchInjector {
    private static final String TAG = "TouchInjector";
    private Thread _thread;
    
    public void touchScreen(int x, int y) {
        final String xy = String.format("%s %s", Integer.valueOf(x), Integer.valueOf(y));
        Thread thread = new Thread(new Runnable() {
            @Override
            public void run() {
                try {
                    String cmd = "input tap " + xy + "\n";
                    TouchInjector.this.executeCommand(cmd);
                    Log.d(TouchInjector.TAG, "터치: " + xy);
                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        });
        this._thread = thread;
        thread.start();
    }
    
    private boolean executeCommand(String command) throws Exception {
        Process process = Runtime.getRuntime().exec("su");
        DataOutputStream os = new DataOutputStream(process.getOutputStream());
        os.writeBytes(command);
        os.writeBytes("exit\n");
        os.flush();
        os.close();
        return process.waitFor() == 0;
    }
}
```

---

### 2. TouchInjector 호출 위치

**전체 코드베이스에서 `touchScreen()` 호출은 단 한 곳에만 존재합니다!**

```java
// DownloadService.java (Line 80-91)
private void runAutoTouch() {
    Log.d(TAG, "설치 버튼 자동 터치 시작.");
    TouchInjector injector = new TouchInjector();
    
    SystemClock.sleep(10000);  // 10초 대기
    Log.d(TAG, "설치 버튼 터치 후 대기");
    Toast.makeText(getApplicationContext(), "설치 버튼 터치 후 대기", 0).show();
    injector.touchScreen(950, 1820);  // ← 첫 번째 터치
    
    SystemClock.sleep(60000);  // 60초 대기
    Log.d(TAG, "실행 버튼 터치");
    Toast.makeText(getApplicationContext(), "실행 버튼 터치 후 대기", 0).show();
    injector.touchScreen(950, 1820);  // ← 두 번째 터치
}
```

---

### 3. runAutoTouch() 호출 흐름

```
1. DownloadService.onHandleIntent()
   └─ APK 다운로드 완료 후

2. runHandler()
   └─ Handler.postDelayed(100ms)

3. runAutoTouch()
   ├─ 10초 대기
   ├─ touchScreen(950, 1820)  // "설치" 버튼 클릭
   ├─ 60초 대기
   └─ touchScreen(950, 1820)  // "실행" 버튼 클릭
```
